Setup for Generate JavaScript Using MATLAB Coder

Introduction

The Generate JavaScript Using MATLAB Coder add-on requires you to install the Emscripten software development kit (SDK) on your computer. The Emscripten SDK uses a group of tools to convert C/C++ code into optimized JavaScript. For more information on Emscripten, see About Emscripten.

Setup

1. Install the this Add-On https://www.mathworks.com/matlabcentral/fileexchange/69973-generatejavascriptusingmatlabcoder. The coder tools require that this add-on be installed into a path without spaces. To set the installation directory of add-ons, in the Home tab, click Environment > Preferences. In the Preferences window, set MATLAB > Add-Ons > Installation Folder to a path without spaces.
2. Install the Emscripten SDK onto your host machine. For instructions on installing the Emscripten SDK, see Download and install. Make note of the root installation directory of the Emscripten SDK, by default this directory is named emsdk/. The esmdk/ folder must be installed into a path without spaces.
3. Add the Emscripten SDK to this Add-Ons external tool registry. Use the following code to set the emsdk folder.
wasm.utilities.ExternalTool.setFolder("Emscripten");
Note: By default, the Node version installed with the Emscripten SDK is used, but a different Node version can be set by explicitly specifying the Node installation folder.
4. Register the Emscripten SDK Toolchain, Emscripten v2.0.NN | gmake (64-bit Windows), and add the Google-V8 Engine target.processor and Web Assembly (WASM) target.LanguageImplementation definitions to MATLAB Coder. To create and register the toolchains, run this command.
wasm.coder.setup(true);
target.add summary:
Objects added to internal database for current MATLAB session:
target.LanguageImplementation "Web Assembly (WASM)"
target.Processor "Google-V8 Engine"
Note: If you do not want to permanently add the Emscripten Toolchain and definitions, use false in the previous command.

Next Steps

>> doc
